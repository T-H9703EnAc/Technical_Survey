# Origin（オリジン）とは

```
http://localhost:4200
https://localhost:4200
http://example.com
https://example.com/
```

上記ようにのURLの「__スキーム(httpやhttps)__」「__ホスト(localhostやexample.com)__」「__ポート(4200)__」の3つの組み合わせが「__オリジン__」です。
